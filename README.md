# \# Calculadora - Fundamentos de Programación

# 

# \## Descripción

# Este proyecto contiene una calculadora básica implementada en \*\*Java\*\* (con IntelliJ IDEA) y en \*\*Pseudocódigo\*\* (con PSeInt).  

# Permite realizar las operaciones de \*\*suma, resta, multiplicación y división\*\* mediante un menú interactivo.

# 

# \## Autora

# \- \*\*Nombre:\*\* Angie Simba

# \- \*\*Materia:\*\* Fundamentos de Programación

# 

# \## Estructura del repositorio

# \- `Suma.java` → Operación de suma en Java

# \- `Resta.java` → Operación de resta en Java

# \- `Multiplicacion.java` → Operación de multiplicación en Java

# \- `Division.java` → Operación de división en Java

# \- `calculadora.psc` → Pseudocódigo con las 4 operaciones en PSeInt

# 

# \## Herramientas usadas

# \- \*\*IntelliJ IDEA\*\* (Java)

# \- \*\*PSeInt\*\* (Pseudocódigo)

# 

# \## Cómo ejecutar

# \### En Java (IntelliJ IDEA)

# 1\. Se abre el archivo `Suma.java`, `Resta.java`, `Multiplicacion.java` o `Division.java`.

# 2\. Y se ejecuta el método `main`.

# 

# \### En PSeInt

# 1\. Se deberá abrir el archivo `calculadora.psc`.

# 2\. Luego ejecutar con F9.
